## Sobre mim e sobre a AllTech

Sou co-fundador da AllTech Websites, um estúdio de desenvolvimento.  
Entregas e presença:
- quatro sites publicados para clientes  
- mais de mil e quinhentos seguidores nas redes sociais  
- perfil verificado no Workana

Trabalho com código organizado, componentes reutilizáveis, responsividade e otimizações de busca.  
Nos repositórios de portfólio apresento escopo, status, datas, imagens e demonstração para avaliação técnica.

## Stack e práticas

Frontend: HTML5, CSS3 e JavaScript  
Backend: PHP 8 ou superior  
Banco de dados: MySQL  
Utilitários: Git e GitHub, Composer, Node para scripts, Hostinger  
Padrões: mobile-first, design system leve, acessibilidade, sessões e validações  
Integrações comuns: QR Code e check-in, PIX, Google Maps e Street View, envio de e-mail por SMTP
